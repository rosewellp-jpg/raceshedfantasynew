# RaceShed Fantasy

Fresh RaceShed Fantasy V16-style React/Vite app for GitHub and Vercel.

## Local setup

```bash
npm install
npm run dev
```

## Vercel settings

- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`
- Install command: `npm install`

## Important repo structure

Your GitHub repo root must contain:

- `package.json`
- `index.html`
- `vite.config.js`
- `src/`
- `public/`

Do not upload a parent folder that contains another nested project folder.
